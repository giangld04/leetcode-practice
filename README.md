# LeetCode Practice

Offline LeetCode problem collection for practice.

## Stats
- **Total Free Problems:** 3191
- **Language:** JavaScript

## Structure
Each problem is in its own folder: `NNNN-problem-slug/`
- `README.md` — Problem description, examples, constraints, hints
- `solution.js` — Code template to fill in
- `test.js` — Test runner with example test cases

## How to Use
1. Pick a problem folder
2. Read the README.md
3. Write your solution in solution.js
4. Run tests: `node test.js`
