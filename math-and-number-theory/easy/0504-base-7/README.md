# 504. Base 7

**Difficulty:** Easy
**Tags:** Math, String
**Link:** https://leetcode.com/problems/base-7/

## Description

Given an integer `num`, return *a string of its **base 7** representation*.

**Example 1:**

```
**Input:** num = 100
**Output:** "202"
```

**Example 2:**

```
**Input:** num = -7
**Output:** "-10"
```

**Constraints:**

	- `-10^7 <= num <= 10^7`
